# 💳 Digital Wallet Application

A modern React Native digital wallet application designed to simplify personal finance management through an intuitive mobile experience. The application enables users to monitor balances, review transaction history, and manage wallet activities through a clean and responsive interface.

## 🚀 Features

* Secure user authentication and account access
* Wallet dashboard with balance overview
* Transaction history tracking
* Send and receive money workflows
* Quick access to recent transactions
* Profile and account management
* Responsive mobile-first design
* Smooth navigation and user experience

## 🛠 Tech Stack

### Frontend

* React Navigation
